# SZU Drcom Login

a helper script to login network in SZU.

Use Selenium webdriver to simulate the manual login prosess.

## Usage:

```bash
# szu-drcom-login -h
usage: szu-drcom-login [-h] location username password

positional arguments:
  location    where are you? (dorm/ta)
  username
  password
```
